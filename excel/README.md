# 📁 Excel Folder

Contains Excel workbooks used for initial data cleaning, validation, and exploratory analysis before building the Power BI model.

| File | Description |

| `credit_card.xlsx` |
Cleaned/validated version of the card transaction data (`credit_card.csv`) used for data quality checks. |


| `customer.xlsx` | 
Cleaned/validated version of the customer demographic data (`cust_add.csv`) used for data quality checks. |


These were used as an intermediate QA step — the final data source for Power BI is the MySQL database built from `/sql` and `/data`.
