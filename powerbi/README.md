
# 📁 Power BI Folder

Contains the final Power BI report file.

| File | Description |
|---|---|
| `credit_card-report.pbix` |
Power BI Desktop file with the full data model (tables joined on `Client_Num`), DAX measures, and two report pages: Credit Card Transaction Report
and Credit Card Customer Report. |

**How to open:** Requires [Power BI Desktop](https://powerbi.microsoft.com/desktop/). 

On opening, update the data source connection to point to your local MySQL instance (see [`/sql`](../sql) for schema), then click **Refresh**.

Dashboard previews are available in [`/images`](../images) and in the main [README](../README.md).
